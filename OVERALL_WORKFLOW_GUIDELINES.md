# Overall Workflow Guidelines and Main Motive

## Main Motive of Project

The Student Skill Development App is designed to:
- Improve student skills
- Track learning progress
- Monitor attendance
- Help admin guide students
- Provide structured skill development

The system acts as a college skill monitoring platform.

## Overall Project Workflow

Student Registration
-> Admin verifies students
-> Admin creates skill courses
-> Admin allocates courses to students
-> Students attend skill sessions
-> Admin marks attendance
-> Assignments are given to students
-> Students complete tasks
-> System calculates progress
-> Leaderboard and reports are generated

## Step-by-Step Guidelines

1. Student Registration
- Only @bitsathy.ac.in email is allowed
- Student record is stored in database

2. Admin Course Creation
- Add skill name
- Set duration
- Save

3. Course Allocation
- Select registered students
- Assign course
- Set from and to date

4. Attendance Tracking
- Admin marks attendance daily
- Status is present or absent

5. Assignment Completion
- Assignment is linked to course
- Students complete tasks

6. Progress Calculation
- Progress is based on attendance and assignments

7. Student Profile Update
- Shows skills
- Shows attendance
- Shows completion

8. Reports and Analytics
- Admin monitors student performance

## Simple Example Workflow

Admin creates Communication Skill
-> Admin allocates to student
-> Admin marks attendance
-> Student completes assignment
-> System updates progress
-> Admin views report

## Final System Flow

Admin controls -> Students learn -> System tracks -> Reports generated

This ensures structured skill development and performance tracking.
