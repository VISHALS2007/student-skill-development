# Icons Documentation

This README file provides information about the icons used in the modern course page UI project.

## Icon Sources

- **Font Awesome**: A popular icon toolkit that provides scalable vector icons that can be customized with CSS. Icons can be found at [Font Awesome](https://fontawesome.com).
- **Material Icons**: A set of icons designed by Google, available for use in web and mobile applications. More information can be found at [Material Icons](https://material.io/resources/icons/).
- **Custom Icons**: Any custom icons created specifically for this project should be documented here, including their design rationale and usage.

## Usage Guidelines

1. **Importing Icons**: Ensure that the icon library is properly imported in the `index.html` file or included in the CSS file.
2. **Accessibility**: Always include `aria-label` attributes for icons that convey important information to ensure accessibility for screen readers.
3. **Sizing**: Icons should be sized appropriately to maintain visual hierarchy. Use CSS classes to adjust size as needed.
4. **Color**: Icons should follow the color scheme of the project. Use CSS to apply colors that align with the overall design.

## Icon List

- **Module Icon**: Represents each module in the course. Use this icon in the module cards.
- **Progress Indicator**: A circular icon that visually represents the progress of the user through the course.
- **Start/Continue Button Icon**: An icon that indicates the action of starting or continuing a module.

## Contribution

If you wish to add new icons or modify existing ones, please follow the project's contribution guidelines and ensure that all icons are consistent with the overall design language of the course page UI.
