# Modern Course Page UI

## Overview
This project is a modern course page UI designed to provide a clean and structured layout for online courses. It features a responsive design that is suitable for both desktop and mobile devices, ensuring an optimal user experience.

## Features
- **Clean Header Layout**: Displays the course title prominently at the top.
- **Course Description**: Provides a brief overview of the course below the title.
- **Module Organization**: Modules are arranged logically, each displayed as a card with:
  - Module number
  - Module title
  - Short description
  - Progress indicator
  - Start/Continue button
- **Sidebar Navigation**: Quick access to modules with highlighted current module.
- **Modern Minimal UI**: Inspired by popular online learning platforms, ensuring a user-friendly interface.

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd modern-course-page-ui
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage
- Open `src/index.html` in a web browser to view the course page.
- The sidebar allows for quick navigation between modules.
- Progress indicators will update as users interact with the course content.
